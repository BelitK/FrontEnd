attribute Selectors

Bu Bölümümüzde CSS Konusunda Önemli İşleve Sahip Attribute Selectors Konusu İşlenecektir.

p[title] title attribute içeren tüm paragrafı seçer.

p[title~="paragraph"] title attribute içerisinde geçen tüm paragrafı seçer.

p[title~="Bilgisayar"] Bilgisayargeçen tüm paragrafı seçer.

p[title="#4 paragrapg"] case-sensitive özelliği bulunmaktadır içerisinde #4 pragraph geçen kısmı seçer.
